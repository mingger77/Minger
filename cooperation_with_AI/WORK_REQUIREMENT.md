# 个人博客风格设计计划

> 总要求：***简单***、***清爽***、***简洁、舒服***

---

## 网站轮廓

1. 整个网站右上角有三个板块，分别为 **主页**、**文章**、**关于**
2. **主页**：
   - 左侧：博客名字 + 博客介绍（可编辑）
   - 右侧：两个区块链接（文章、关于）+ 一个深米色方块用于写其他文字
   - 底部右侧小字 **authored by mingger77**

---

## 功能要求

1. **编辑主页**
   - 博客名字
   - 博客介绍

2. **编辑文章**
   - 文章创作时间
   - 文章标题
   - 文章正文

3. **编辑关于**
   - 博客个人信息
   - 博客联系方式
   - 博客内容拓展

4. **自定义网站图标**

5. **提供 `GUIDE.md` 使用向导**

---

## 美学风格

参考 **Anthropic** 风格。

| 方面 | 当前实现 |
|------|---------|
| 主体颜色 | 浅米色 `#faf5eb` |
| 背景点缀 | 无（已去除） |
| 顶栏/底栏色 | 深米黄色 `#c4a86a` |
| 顶栏/底栏文字颜色 | 黑色 |
| 顶栏/底栏文字右距 | 固定 `24px` |
| 文字部分的底色（/posts 列表） | 每个 item 独立深米黄色 `#ece0c8` 分隔块 |
| 文字部分的底色（文章详情/关于） | 一整块连续深米黄色 `#ece0c8` |

---

## 鸽子设计

| 要求 | 当前实现 |
|------|---------|
| 出现位置 | 所有页面背景 |
| 朝向 | 从右往左看（头在右侧） |
| 大小 | 占据网站中心约 1/2 |
| 定位 | `position: fixed`，居中 |
| 颜色 | 深米色 `#d4bf90`（与浅米色背景形成对比） |
| 透明度 | `0.35` |
| 图层顺序 | 在文字之下（`z-index: 5`），在主页跳转链接按钮之上 |

---

## 主页布局

```
+-------------------------------------------+
|  顶栏（深米黄色，文字黑色，右距 24px）      |
+-------------------------------------------+
|                                            |
|  +-- 左侧 45% --+  +-- 右侧 45% ---------+|
|  |               |  | [ 文章 → ] 区块链接   ||
|  | 博客名字      |  | [ 关于 → ] 区块链接   ||
|  | (6rem)       |  | [深米色文字方块]      ||
|  | 博客介绍      |  |                      ||
|  | (2.1rem)     |  +----------------------+|
|  +---------------+                         |
|                                             |
|   鸽子（fixed 居中，z-index:5，在按钮之上）   |
+-------------------------------------------+
|  底栏（深米黄色，authored by mingger77）   |
+-------------------------------------------+
```

---

## 尺寸与响应式

| 元素 | 默认 | 移动端（≤768px） |
|------|------|-----------------|
| 博客名字 | `6rem`，`z-index: 10` | `3rem` |
| 博客介绍 | `2.1rem`，`z-index: 10` | `1.3rem` |
| 主页左右 | `flex: 0 0 45%` | `flex: none`（堆叠） |
| 文章/关于内容框 | `max-width: max(640px, 66vw)` | 100% |
| 顶/底栏 | `padding-right: 24px` | 同上 |
| 鸽子 | `50vw`，max `560px` | `80vw` |
